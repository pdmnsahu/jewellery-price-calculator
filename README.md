# Jewellery Price Calculator

A jewellery rate calculator for **22K Gold**, **18K Gold**, and **Silver**, with making charges, GST calculation, and a reverse budget calculator.

---

## Features

- Live rate input for 22K Gold, 18K Gold, and Silver (per 10g)
- Making charges — percentage + fixed ₹/g per metal
- Auto-calculated GST (3%) and final customer price
- Reverse calculator — enter a budget to see how many grams of 22K gold you get
- State persistence via `localStorage` — rates survive page refresh
